# Contributing!

First off, thank you for thinking about contributions. Extra help is always appreciated! Remember to follow our policies and conduct.
## Issues

You can make an issue at https://github.com/noobieyuh/Spot-The-Phish/issues.

## Contribute (Pull Requests)

Create a fork of this repository and make your changes, afterwards, create a pull request and we will review it! Make sure it follows our policies.

### Commit messages

When creating commit messages, use the formats at https://udacity.github.io/git-styleguide/!

- `feat`: A new feature
- `fix`: A bug fix
- `docs`: Changes to documentation
- `style`: Formatting, missing semicolons, etc.; no code change
- `refactor`: Refactoring production code
- `test`: Adding tests, refactoring test; no production code change
- `chore`: Updating build tasks, package manager configs, etc.; no production code change

You can create or use a tag not listed here as long as it makes sense! For example:
- `typo`: Corrects a spelling error
